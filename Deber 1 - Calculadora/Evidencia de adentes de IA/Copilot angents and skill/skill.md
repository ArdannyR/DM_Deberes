# Guía para Ejecutar y Probar la Calculadora EPN

Este documento proporciona instrucciones paso a paso sobre cómo ejecutar y probar el proyecto "Calculadora EPN" utilizando la extensión Live Server.

## Paso 1: Instalar la Extensión Live Server

1. Abre tu editor de código.
2. Dirígete a la sección de extensiones. Puedes hacerlo haciendo clic en el ícono de extensiones en la barra lateral o utilizando el atajo de teclado correspondiente.
3. En la barra de búsqueda, escribe "Live Server".
4. Selecciona la extensión "Live Server" de Ritwick Dey.
5. Haz clic en el botón "Instalar".

## Paso 2: Abrir la Carpeta del Proyecto

1. En tu editor de código, selecciona "Archivo" en la barra de menú.
2. Haz clic en "Abrir Carpeta".
3. Navega hasta la carpeta "Deber 1 - Calculadora" y selecciónala.
4. Haz clic en "Abrir".

## Paso 3: Iniciar el Servidor en Vivo

1. Una vez que la carpeta del proyecto esté abierta, haz clic derecho en el archivo `index.html` en el explorador de archivos.
2. Selecciona la opción "Open with Live Server" (Abrir con Live Server).
3. Esto abrirá tu navegador predeterminado y cargará la calculadora.

## Paso 4: Interactuar con la Calculadora

1. En la interfaz de la calculadora, verás campos de entrada para las notas del primer y segundo bimestre, así como para la nota del supletorio.
2. Ingresa las notas en los campos correspondientes.
3. Haz clic en el botón "CALCULAR" para obtener los resultados.
4. Observa cómo se actualizan los mensajes y los resultados en tiempo real según las notas ingresadas.

## Paso 5: Probar Diferentes Escenarios

1. Prueba diferentes combinaciones de notas para verificar que la lógica de cálculo funcione correctamente.
2. Asegúrate de que los mensajes de éxito, error y advertencia se muestren adecuadamente según las reglas de evaluación.

Siguiendo estos pasos, podrás ejecutar y probar la "Calculadora EPN" de manera efectiva utilizando la extensión Live Server.
